# Task 14 - Data Cluster Parameter Prep

## About

In this task, I prepared the Penguin dataset for clustering. First, I checked the dataset, selected useful numerical features, scaled the data using StandardScaler, and then used K-Means clustering. I also used the Elbow Method and Silhouette Score to find the best number of clusters.

---

## Dataset

Dataset: Penguin Dataset

Features used for clustering:

- Bill Length
- Bill Depth
- Flipper Length
- Body Mass
